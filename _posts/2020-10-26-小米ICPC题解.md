## 题意
有一个n*m的二维矩阵，每个格子可以向上下左右之一的方向走，问有多少个格子可以间接或直接走出矩阵
## 思路
二维并查集裸题，最后遍历所有格子，如果父节点在矩阵外就计数
## 代码
```c++
#include <bits/stdc++.h>
#define inf 0x7f7f7f7f
#define ms(a) memset(a, 0, sizeof(a))
#define repu(i, a, b) for (int i = a; i < b; i++)
#define repd(i, a, b) for (int i = a; i > b; i--)
using namespace std;
typedef long long ll;
typedef long double ld;

const int M = int(1e3) + 5;
const int mod = int(1e9) + 7;

struct node {
    int x, y;
    bool operator==(const node& a) {
        if (a.x == x && a.y == y) {
            return true;
        }
        return false;
    }
};
node fa[M][M];
void init() {
    repu(i, 0, M) {
        repu(j, 0, M) { fa[i][j] = {i, j}; }
    }
}
node find(node x) {
    if (fa[x.x][x.y] == x) {
        return x;
    }
    return fa[x.x][x.y] = find(fa[x.x][x.y]);
}
void merge(node x, node y) {
    node xx = find(x);
    node yy = find(y);

    fa[xx.x][xx.y] = yy;
}

void solve() {
    int n, m;
    cin >> n >> m;

    init();

    repu(i, 1, n + 1) {
        repu(j, 1, m + 1) {
            char c;
            cin >> c;
            if (c == 'W') {
                merge({i, j}, {i - 1, j});
            }
            if (c == 'A') {
                merge({i, j}, {i, j - 1});
            }
            if (c == 'S') {
                merge({i, j}, {i + 1, j});
            }
            if (c == 'D') {
                merge({i, j}, {i, j + 1});
            }
        }
    }

    // repu(i, 1, n + 1) {
    //     repu(j, 1, m + 1) { cout << fa[i][j].x << "," << fa[i][j].y << " "; }
    //     cout << endl;
    // }

    ll ans = 0;
    repu(i, 1, n + 1) {
        repu(j, 1, m + 1) {
            node f = find({i, j});
            
            if (!((f.x >= 1 && f.x <= n) && (f.y >= 1 && f.y <= m))) {
                ans++;
            }
        }
    }

    cout << ans << endl;
}
int main() {
    ios::sync_with_stdio(false);
    cin.tie(0);

    // int t;
    // cin >> t;
    // while (t--) {

    // }
    solve();
    return 0;
}
```
